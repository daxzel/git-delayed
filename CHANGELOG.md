# Changelog

## [0.1.0] - 2024-11-02

First release.

- Schedule commits and pushes separately
- Time formats: relative (+10 hours), named days (Monday), absolute (2025-11-04 09:00)
- Works as git plugin (`git delayed`)
- Auto-retry on failure (every 10 minutes)
- Background daemon via launchd/systemd
- macOS and Linux support
- Homebrew formula ready
